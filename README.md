# chamsae monorepo

참새작 점수계산기를 웹(Next.js)과 모바일(React Native)에서 함께 관리하는 모노레포입니다.

## Packages

- `packages/core`: 참새작 코어 로직
- `packages/web-next`: 참새작 점수계산기 Next.js 앱
- `packages/mobile-rn`: 참새작 점수계산기 React Native(Expo) 앱

## Install

```bash
npm install
```

## Run Web (Next.js)

```bash
npm run dev:web
```

## Run Mobile (React Native)

```bash
npm run dev:mobile
```

## Typecheck

```bash
npm run typecheck
```

## Build Web

```bash
npm run build:web
npm run start:web
```
